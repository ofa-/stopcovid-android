StopCovid Android
=================

_Extended_ StopCovid app, featuring:

- notifications on "ping"
- proximity info screen
- ui beautifications

This app is a fork of upstream [gitlab.inria.fr][upstream]
`stopcovid-android` and is compatible with [StopCovid][StopCovid].
